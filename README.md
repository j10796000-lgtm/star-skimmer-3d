# Star Skimmer 3D

A fast, colorful browser game built with Three.js. Steer the skimmer, collect stars,
hit boost rings, dodge spinners, and chase bigger combo scores.

## Play locally

Serve the folder with any static server, then open the local URL in a browser.

## Controls

- `A` / `D` or arrow keys: steer
- `Space`: boost

## Publish

This is a static site and works well on GitHub Pages.
